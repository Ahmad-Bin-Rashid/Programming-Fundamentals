# Programming Fundamentals Coursework

This repository contains my coursework, lab tasks, and programming assignments for the **Programming Fundamentals** course.


## Repository Structure

The workspace is organized into several directories based on the type of assignments:

- **`PFLab */`**: Weekly Programming Fundamentals Lab tasks. Contains hands-on practice programs in C++ and Python.
- **`PD-*/`**: "Programming Day" tasks. These are more comprehensive assignments and problems.
- **`Business Application/`**: A larger project implementation, containing versions in both C++ and Python.

## Languages Used
- **C++**: The primary language used for most of the core logic and problem-solving tasks.
- **Python**: Used in some later labs.

## How to Run

### C++ Files
To compile and run any of the C++ files, you can use `g++`. For example:
```bash
g++ PFLab\ 2/circle.cpp -o circle
./circle
```

### Python Files
To run the Python scripts:
```bash
python PFLab\ 14/task3.py
```
